# KOEditor
This is a editing tool to quickly load, build, and edit knowledge objects.  

## Quick Start

  Install dependencies:
```bash
$ npm install
```

  Start the server:
```bash
$ node index.js
```

  Configuration  :
```
/cfg/sv.koeditor.json
```

  Open in browser  :
```
http://localhost:8066
```
  with default activator server url 
```
http://localhost:8066#http://activator.server.url/stack/ 
```
